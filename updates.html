LATEST RELEASE 2.0: UX OVERHAUL

  • Unified the command-center HUD theme across Ban Checker, About, and Games.
  • Navigation and CTAs restyled for clearer hierarchy and consistent glow states.

  • Version history now loads collapsed for a faster overview.

Pre-Web / EXE Era

v0.1 - v0.5: Early EXE builds limited to Steam; API was bundled in the client and not shareable.

v0.6: First EXE supporting all platforms, with the API still embedded and insecure.

v0.7: Introduced basic color feedback for ban results in the EXE.

v0.8: Backend created and hosted on Render to separate API from the client.

v0.9: Community EXE test build; Render kept warm, flagged by Windows Defender.

Web Era

v1.0: Pivot to a GitHub-hosted website with a functional PC ban checker.

v1.1: Multi-platform support (Steam, Xbox, PSN, Kakao) introduced.

v1.2: Watchlist creation added (initially PC only).

v1.2.1: Watchlist creation expanded to all platforms (Steam, Xbox, PSN, Kakao).

v1.3: UX improvements across core flows.

v1.4: Search by account ID added.

v1.5: Security fixes delivered (thanks LOLhpO!).

v1.6: Watchlist now flags when a player changes their name.
